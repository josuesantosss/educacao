Atalhos do Windows

Copia o conteúdo selecionado.

Passo 1: Selecione o texto a ser copiado;

Passo 2: Segure a tecla Ctrl e pressione a letra "C".

Dica: Você pode copiar arquivos, pastas e outros itens da mesma maneira!

Cola o conteúdo que foi copiado.

Passo 1: Coloque o cursor aonde o texto será colado;

Passo 2: Segure a tecla Ctrl e pressione a letra "V" e o texto será colado.

Observação: O conteúdo permanece na área de transferência até ser substituído.

Recorta o conteúdo selecionado.

Passo 1: Selecione o texto a ser recortado;

Passo 2: Segure a tecla Ctrl e pressione a letra "X" e o texto será movido para o novo local.

Diferença: Copiar duplica o conteúdo, recortar move o conteúdo.

Desfaz a última ação.

Pressione Ctrl+Z para desfazer a última alteração feita.

Dica: Em muitos programas, você pode pressionar várias vezes para desfazer múltiplas ações!

Abre as opções de projeção para conectar a um projetor ou segunda tela.

Opções disponíveis:

  • Somente tela do computador
  • Duplicar
  • Estender
  • Somente segunda tela

Abre o Menu Iniciar.

Pressione a tecla do Windows (⊞) para abrir o menu iniciar.

Dica: Comece a digitar imediatamente para pesquisar programas e arquivos!

Captura a tela inteira e copia para a área de transferência.

Passo 1: Pressione a tecla Print Screen (PrtScn);

Passo 2: Abra um programa como Paint ou Word e cole (Ctrl+V) a imagem.

Dicas avançadas:

  • Alt+PrtScn captura apenas a janela ativa
  • Win+Shift+S abre a ferramenta de recorte

Alterna entre janelas abertas.

Passo 1: Mantenha pressionada a tecla Alt;

Passo 2: Pressione Tab para navegar entre as janelas;

Passo 3: Solte as teclas para selecionar a janela.

Dica: Win+Tab mostra todas as janelas em uma vista mais detalhada.

Abre diretamente o Gerenciador de Tarefas.

Pressione Ctrl+Shift+Esc para abrir o Gerenciador de Tarefas.

Útil para:

  • Verificar desempenho do sistema
  • Fechar programas não respondendo
